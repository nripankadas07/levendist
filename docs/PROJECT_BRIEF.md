        # Project Brief

        levendist exists to solve a narrow, inspectable developer-tooling problem:
        String-distance metrics: Levenshtein, Damerau-Levenshtein, Hamming, Jaro, Jaro-Winkler, LCS. Zero dependencies.

        primitive, a benchmark surface, or a concrete local workflow.
